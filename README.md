HIII, this macropad is fully self/custom made. It has 6 keys and uses a Seeed XIAO RP2040 for software it uses QMK.

the parts are 6 through-hole 1N4148 Diodes, 6 MX-Style switches, 1 unsoldered Seeed XIAO RP2040, 4x M3x16mm screws, 4x M3x5mx4mm heatset inserts and the case is 3d printed.
